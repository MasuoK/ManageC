javaのファイルは大文字から始める
public class は１つまで(そのプログラムと同じ名前のクラス)

コンストラクタ：インスタンスが生成された時に呼ばれる特別なメソッド？
引数とかを初期化するのに使う
何も書かないとデフォルトコンストラクタといって引数なしのコンストラクタが実行される
明示的に書く必要があったりなかったりするので要確認


abstract がついたやつは抽象クラス(抽象メソッド:abstractがついたメソッドを１つ以上持つクラス)
抽象メソッドはプロトタイプ宣言のような感じで関数名と型と引数だけ書く。

抽象クラスからはインスタンスは生成できないことに注意(Car car = new Car() みたいにできない)
インスタンスを作るときは抽象クラスを継承をしたサブクラスを使う
抽象クラスを継承したサブクラスではabstractで書かれた関数を@Overrideをつけて
中身を実装して使う。
